# SoundDeck v1.0.0

Initial polished release.

## Features

- Local audio soundboard
- Automatic VB-Cable relay detection
- Discord/OBS microphone routing
- Optional mic passthrough
- Optional local monitoring
- Optional mic monitoring
- Multiple audio folders
- Queue and playlists
- Autoplay next
- Loop song
- Loop playlist
- Dark/light themes
- FFmpeg decoding for common formats

## Notes

FFmpeg binaries are not included in the source repository. Add `ffmpeg.exe` and `ffprobe.exe` to a local `ffmpeg` folder before building from source.
